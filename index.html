Изучите Основы

Три кита разработки под Web-приложения — это HTML, CSS и JavaScript, с которых мы и начинаем.

Разберемся, как работает Web, как работает клиент-серверная модель. Изучим структуру HTML-документа, что такое теги и какие они есть, мета-теги, атрибуты, как добавлять изображения, создавать формы. Изучим блочную модель, позиционирование, каскадирование стилей, специфичность селекторов, псевдоэлементы, адаптивную верстку (для компьютеров, планшетов и телефонов). Научимся верстать современные макеты с помощью Flexbox и Grid.

Подробнее

Изучите React

React имеет огромную популярность среди разработчиков и компаний, прост для понимания начинающими специалистами.

React популяризировал совершенно новую архитектуру веб-приложений под названием «Одностраничное приложение». Ранее веб-страница загружалась с сервера, и любой щелчок приводил к выполнению нового запроса к серверу, и браузер загружал новую страницу. Теперь же любое дальнейшее взаимодействие с приложением загружает только необходимые данные или выполняет действие на сервере.
